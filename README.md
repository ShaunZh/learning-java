### 方法的执行机制

1. 当执行方法时，会在栈中开辟一个独立的内存空间(栈空间)，并将当前 PC 指针保存，执行方法
2. 当方法执行完毕后，会将保存的 PC 指针地址取出，并将 PC 指针指向原保存 PC 指针，用于返回到方法执行前的位置
3. 释放方法占用的栈空间
4. 结束
